Delphi Front End Transpiler

This documentation delves into all aspects of the front end transpiler tool.

It starts with parsing Delphi VCL components, represented by the .dfm files. Going into details of designing the DFM grammar, which provides a solid foundation and informs the reader on techniques used for expanding said grammar, if necessary in the future.

Once the foundations have been established, a refined and ready-to-use DFM grammar will be looked at, and will be parsed using ANTLR4 in later sections.
